This project was developed in my third bachelor year in collaboration with a startup that was then called Furnify (now [NurbanSpace](https://www.nurbanspace.com/)).

It is a 3D room configuration application built with [React Three Fiber](https://docs.pmnd.rs/react-three-fiber) and [Vite](https://vitejs.dev/) on [Node.js](https://nodejs.org/). Users can design and configure a room through a guided questionnaire and place modular furniture components in an interactive 3D environment.

The project focused on real-time 3D interaction in the browser, user-driven configuration logic, and translating functional requirements into a spatial design tool. It was developed in a team setting with iterative feedback from the startup.
