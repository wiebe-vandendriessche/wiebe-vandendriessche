Master's Computer Graphics course project, developed in collaboration with Walter Di Sabatino. The goal was to visualize cardiovascular data in the browser using [VTK.js](https://kitware.github.io/vtk-js/).

The project implements multiple 3D visualization techniques such as virtual endoscopy, cross-sectional views, and maximum intensity projection (MIP). It enables interactive exploration of vascular structures, including navigation through vessel branches in real time.

The application is built with [VTK.js](https://kitware.github.io/vtk-js/) and [Vite](https://vitejs.dev/) and runs entirely in the browser, allowing efficient and interactive inspection of complex medical datasets.
