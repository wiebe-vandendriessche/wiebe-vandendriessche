This project was created as a learning exercise to get familiar with [Go](https://go.dev/) and REST API development.

It is a lightweight vulnerability scanning prototype that runs scans on targets via an HTTP API, stores results in [SQLite](https://www.sqlite.org/), and exposes endpoints to track and share scan results.

The tool was initially explored in the context of the [CRACY project](https://cra-cy.eu/), but it was later kept as a personal learning project since it did not introduce enough novel research contributions for integration into the CRACY framework.
